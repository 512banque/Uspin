Spin classique

Choisit au hasard une possibilité.

{bonjour|hello|hi}

donnera :

bonjour

Spin suivi

Choisit au hasard un numéro et garde le même numéro tout le long du texte.

Hi, my name is
#1#Mark#/1##2#Carla#/2##3#James#/3#
And I love #1#apples#/1##2#pickles#/2##3#cheese#/3#

donnera :

Hi, my name is
Mark
And I love apples

Tag

Crée une balise html. Fonctionne par paires.

[tag=h1]Mark[/tag]
[tag=strong]I like [tag=u]drinking[/tag] coke[/tag]
[tag={h1|h2|h3|p}]{Yet |}Another title[/tag]

donnera :

<h1>Mark</h1>
<strong>I like <u>drinking</u> coke</strong>.
<h3>Another title</h3>

List

Crée une liste, en la mélangeant et en ajoutant/supprimant des éléments, avec l'argument passé en séparateur. Si pas d'argument la fonction en génère un au hasard. Les éléments de la liste doivent être séparés par un point-virgule.

[list=li]
this is my first element;
this is my second element;
this is my third element;
this is my fourth element
[/list]

[list=,]
this is my first element;
this is my second element;
this is my third element;
this is my fourth element
[/list]

[list==>]
this is my first element;
this is my second element;
this is my third element;
this is my fourth element
[/list]

[list=+]
this is my first element;
this is my second element;
this is my third element;
this is my fourth element
[/list]

donnera :

<ul>
<li>this is my third element</li>
<li>this is my first element</li>
<li>this is my fourth element</li>
</ul>

this is my fourth element, this is my first element, this is my second element, this is my third element

=>this is my second element
<br>=>this is my third element
<br>=>this is my first element
<br>=>this is my fourth element

+this is my fourth element
<br>+this is my third element
<br>+this is my second element

Size

Définit la taille du texte entre les tags.

[size=15]my text[/size] is the best

donnera :

<span style="font-size:15px">my text is the best</span>

Img

Insère une image ayant comme url l'argument entre les tags.

[img]http://mysite/mypic.jpg[/img] is the best

donnera :

<img src="http://mysite/mypic.jpg" />

Flickr

Aspire une image aléatoire de flickr trouvée avec le mot clé entre les tags.

[flickr]car[/flickr]
[flickr]{car|bike}[/flickr]

donnera :

<img src="http://farm5.static.flickr.com/4006/4619399567_90a7d8dc6f.jpg"><br>DSC_8364
